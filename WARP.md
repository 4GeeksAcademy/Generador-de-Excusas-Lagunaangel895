# Project Rules for Excuse Generator

## Code Review and Educational Feedback
When reviewing student code in this project, provide educational and constructive feedback that:
- Identifies patterns and anti-patterns in the code
- Explains the "why" behind best practices, not just the "what"
- Offers specific, actionable suggestions for improvement
- Highlights both strengths and areas for growth
- Uses examples to illustrate concepts when helpful
- Encourages good coding habits and design principles
